## 三、关键改动说明

1. Pipeline Rail
- 由 `flex + min-width + overflow-x-auto` 改为 `grid` 等分布局（`sm:grid-cols-4` / `xl:grid-cols-8`）。
- 阶段卡片高度从 `min-h-[116px]` 调整为 `min-h-[96px]`，内边距同步收紧。
- 阶段箭头改为绝对定位，保持常显。

2. 双四栏主区
- 两行主网格均将断点从 `2xl` 下调至 `xl`，保证 1440px 屏宽下即进入四栏布局。
- 行间距从 `gap-6` 调整到 `gap-5`，提升与效果图的紧凑度一致性。

3. 产物预览
- 卡片网格改为固定 `grid-cols-2`，保持稳定 2x2 呈现。
- 缩略图区高度改为 `h-32`。
- Manim 卡片播放按钮调整为中轴居中并适度放大。

4. 全局面板风格
- `Panel` 圆角改为 `rounded-[24px]`，与审查建议一致。
